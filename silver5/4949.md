### boj 4949
```
import sys
input = sys.stdin.readline

check = ['(', ')', '[', ']']
inputs = input().rstrip()
while inputs != '.':
    stack = []

    for i in inputs:
        
        if i in check:
            if len(stack) == 0:
                stack.append(i)
                continue
            if stack[-1] == '(': # pop
                if i == ')':
                    stack.pop()
                    continue

            elif stack[-1] == '[': # pop
                if i == ']':
                    stack.pop()
                    continue
            stack.append(i) # append

    if len(stack) == 0: # check
        print("yes")
    else:
        print('no')

    inputs = input().rstrip()
```

- AB 문제와 다른점
    - ‘)’나 ‘]’가 들어왔는데, 스택의 가장 위가 ‘(’나 ‘[’가 아닐 경우나 스택이 비었을 경우 바로 False 리턴 가능함
